# Установка
___
# Получаем токен [VKAdmin.](https://vk.cc/9NCoPi)
+ Листаем в самый низ, нажимаем [Разрешить].
+ Копируем то-что находиться в URL адресе страницы

# Переходим в личные сообщения с Агентами.
+ Пишем - зарегистрируй
+  + Получаем - :white_check_mark: «SKRLP Module» Token VKA updated.

**Проверяем** - ```{CPREFIX}``` инфо
___
# Получаем токен [Sakura](vk.com/mail?peer={}).
___
+ Отключаем 2Хфакторную антентификацию.
+ Вспоминаем логин и пароль от страницы.

**Переходим в диалог избранный** 
+ Пишем - ```.к токен {ENTER} {LOGIN} {ENTER} {PASS}```
+ + Получаем -  :white_check_mark: «SKRLP Module» Token SKR updated.

**Проверяем** - ```{CPREFIX}``` инфо
+ :arrow_up: [Навигация](https://github.com/sssakurasss/sssakurasss/edit/main/README.md#%D0%BD%D0%B0%D0%B2%D0%B8%D0%B3%D0%B0%D1%86%D0%B8%D1%8F)

# Функции 

**Информация аккаунта**
+ ```{CPREFIX}``` инфо - Профиль в боте
___
**Статистика аккаунта**
+ ```{CPREFIX}``` - стата - Статистика аккаунта
___
**Прочитать сообщение**
+ ```{CPREFIX}``` прочитать ```все/лс/группы/беседы``` ```{NONE}/подробно``` - Пометит сообщения как прочитанные
___
**Текст аудио**
+ ```{CPREFIX}``` слова - Вернет текст аудиозаписи
___
**Текст голосовое**
+ ```{CPREFIX}``` озвучь ```{ENTER}``` ```{TEXT}``` - Озвучит текст
___
**Стикеры**
+ ```{CPREFIX}``` стики - Вернет стикеры пользователя
___
**Пролайкать**
+ ```{CPREFIX}``` пролайкать ```{USER}``` - Пролайкает страницу
___
**Дата регистрации**
+ ```{CPREFIX}``` датарег ```{USER}``` - Отобразиит дату регистрацию пользователя в VK
___
**Включить уведомления**
+ ```{CPREFIX}``` +ув ```{USER}``` - Включит уведомления
___
**Выключить уведомления**
+ ```{CPREFIX}``` -ув ```{USER}``` - Выключит уведомления
___
**Добавить в друзья**
+ ```{CPREFIX}``` +др ```{USER}``` - Добавит в друзья
___
**Удалить из друзья**
+ ```{CPREFIX}``` -др ```{USER}``` - Удалит из друзья
___
**Добавить в доверенные**
+ ```{CPREFIX}``` +дов ```{USER}``` - Добавит в доверенные
___
**Удалить из доверенных**
+ ```{CPREFIX}``` -дов ```{USER}``` - Удалит из доверенных
___
**Добавить в чс**
+ ```{CPREFIX}``` +чс ```{USER}``` - Добавит в чс
___
**Удалить с чс**
+ ```{CPREFIX}``` -чс ```{USER}``` - Удалит с чс
___
**Выдать админа**
+ ```{CPREFIX}``` +адм ```{USER}``` - Выдаст права администратора в чате
___
**Забрать админа**
+ ```{CPREFIX}``` -адм ```{USER}``` - Заберет права администратора в чате
___
**Удалить сообщения**
+ ```{CPREFIX}``` дд ```{NUMBER}``` - Удалит сообщения
___
**Кикнуть с чата**
+ ```{CPREFIX}```кик ```{USER}``` - Удалит пользователя с чата
___
**Добавить в чат**
+ ```{CPREFIX}``` добавить ```{USER}```- Добавить пользователя в чат
___
**Вступить в группу**
+ ```{CPREFIX}``` группа ```{USER}``` - Вступит в группу
___
**Выйти с чата**
+ ```{CPREFIX}``` выйти - Выйдет из беседы
___
**Редактирование смс**
+ ```{CPREFIX}``` эдит - Редактирует смс ( Функция в разработке )
___
**Демотиватор**
+ ```{CPREFIX}``` дем ```{ENTER}``` ```{TEXT}``` фото - Демотивирует текст
___
**Чистка собак**
+ ```{CPREFIX}``` !собак - Чистит замороженные/удалённые страницы из друзей
___
**Чистка групп**
+ ```{CPREFIX}``` !группы - Чистит группы на которые вы подписаные, но не описывается где вы администратор/создатель
___
**Чистка истоков**
+ ```{CPREFIX}``` !исток - Чистит источники, на кого вы поставили уведомления. Удаляет по: 100
___
**Чистка друзей**
+ ```{CPREFIX}``` !др - Чистит друзей, удаляет по 100
___
**Чистка сообщений**
+ ```{CPREFIX}``` !смс - Чистит личные сообщения
___
**Чистка стены**
+ ```{CPREFIX}``` !стена - Чистит страницу от постов
___
**Шаги**
+ ```{CPREFIX}``` шаги - Проходит 80,000км и 50,000 метров. 
___
**Айди**
+ ```{CPREFIX}``` ид - Показывает цифровой айди пользователя.
